# Лабораторна робота 1: Класифікація (Data Mining)

## Датасет
**Adult Census Income** — прогнозування річного доходу (<=50K / >50K) на основі даних перепису населення США 1994 року.

## Структура проекту

| Файл | Опис |
|---|---|
| `01_preprocessing.ipynb` | Завантаження, EDA, обробка пропусків/дублікатів/викидів, відбір ознак, кодування |
| `02_oneR.ipynb` | Класифікатор 1-Rule (OneR) |
| `03_naive_bayes.ipynb` | Класифікатор Naive Bayes (GaussianNB) |
| `04_decision_tree.ipynb` | Класифікатор Decision Tree |
| `05_knn.ipynb` | Класифікатор kNN (k-найближчих сусідів) |
| `06_comparison.ipynb` | Порівняльний аналіз всіх класифікаторів |

## Порядок запуску

1. Встановити залежності: `pip install -r requirements.txt`
2. Запустити `01_preprocessing.ipynb` (створює `data_processed.csv`)
3. Запустити ноутбуки 02-05 у будь-якому порядку
4. Запустити `06_comparison.ipynb` для фінального порівняння

## Алгоритми

- **1-Rule** — вибирає один найкращий атрибут для класифікації
- **Naive Bayes** — ймовірнісний класифікатор на основі теореми Баєса
- **Decision Tree** — рекурсивне розбиття за інформаційним приростом
- **kNN** — класифікація за k найближчими сусідами
